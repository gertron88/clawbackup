## Submission Strategy

### For Infrastructure-Focused Hackathons:
1. Emphasize **developer experience**
2. Show **security model** (client-side encryption)
3. Demo **real-world usage** (backup/restore)
4. Highlight **open source** nature

### For General AI Hackathons:
1. Frame as "AI agent lifecycle management"
2. Show how it enables **reliable AI systems**
3. Demo with an actual AI agent
4. Emphasize **disaster recovery**
